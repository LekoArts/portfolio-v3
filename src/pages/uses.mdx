## Editor & Terminal

- [Visual Studio Code](https://code.visualstudio.com/)
- [JetBrains Mono](https://www.jetbrains.com/lp/mono/) as VSCode font
- [FiraCode](https://github.com/tonsky/FiraCode) as terminal font
- [zsh](https://www.zsh.org/) + [oh-my-zsh](https://ohmyz.sh/)
  - [spaceship](https://github.com/denysdovhan/spaceship-prompt) as zsh prompt
  - Plugins: `plugins=(git zsh-autosuggestions zsh-syntax-highlighting zsh-dircolors-solarized history gh rust ohmyzsh-full-autoupdate zsh-tab-title z)`
- [Windows Terminal](https://github.com/microsoft/terminal) & [iTerm2](https://iterm2.com/)
- [bat](https://github.com/sharkdp/bat) as a `cat` replacement
- [eza](/garden/replacing-ls-with-eza/) as an `ls` replacement
- [delta](https://github.com/dandavison/delta) as a syntax-highlighting pager for git, diff, and grep output
- [fnm](https://github.com/Schniz/fnm) as Node.js version manager
- [fzf](https://github.com/junegunn/fzf), [fd](https://github.com/sharkdp/fd), [rg](https://github.com/BurntSushi/ripgrep)
- On Windows 10 I use [WSL 2 (Ubuntu)](https://ubuntu.com/wsl) as package manager for developing, on MacOS I use [MacPorts](https://www.macports.org/)
